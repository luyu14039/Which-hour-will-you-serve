# Audio Assets Required

Please place the following audio files in this directory (`client/public/sounds/`):

- `ambient.mp3` - Background ambience (looping recommended)
- `click.mp3` - Button click sound
- `hover.mp3` - Button hover sound (short, subtle)
- `flip.mp3` - Card flip sound
- `reveal.mp3` - Result reveal sound (impactful)

The application is configured to look for these specific filenames.
